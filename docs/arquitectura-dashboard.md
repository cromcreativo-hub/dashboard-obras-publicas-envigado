# Dashboard Obras Públicas Envigado

## Objetivo
Crear un dashboard dinámico y escalable para seguimiento de indicadores, metas y resultados de la Secretaría de Obras Públicas de Envigado.

## Principios

- Módulos configurables
- Indicadores dinámicos
- Unidades configurables
- Metas por vigencia
- Carga Excel
- Dashboard responsive
- Buscador global
- Usuarios y permisos
- Reportes descargables
- Escalabilidad
